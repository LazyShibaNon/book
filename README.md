# book

使用者開啟程式，點選“收集資料按鈕後”可得到: <br>
a. 一個新的視窗，回傳使用者資料收集的結果<br>
b. 與程式連結的Line會推播通知網頁資料有無更新<br>
c. 一個用來判斷網頁資料是否更新的“雜湊值”txt檔<br>
d. 排行榜書籍資料的csv檔<br>
